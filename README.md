# Study Helper

موقع ويب يساعد الطلاب و الموظفين على تنظيم عملهم و دراستهم من خلال الخرائط الذهنية و جداول الطلاب ومراجعتهم بطريقة ذكية لتسهيل عمل اي فرد بالمجتمع

# الميزات
جداول المواعيد والمحاضرات
عرض الأخبار والإعلانات الجامعية
تحميل ومشاركة الملفات الأكاديمية
إمكانية البحث السريع عن محتوى الدروس
متجاوب على جميع الأجهزة
تحميل الخرائط الذهنية ب الشكل الذي يلهمك
مشاركة نصائح وملخصات 

# التقنيات المستخدمة
Frontend: HTML, CSS, JavaScript, أو مكتبة حديثة  React/Vue

Backend: Node.js مع Express، أو Django/Flask

قاعدة بياناتMySQL، PostgreSQL، أو MongoDB

Responsive Design: CSS Grid/Flexbox أو مكتبات مثل Tailwind CSS/Bootstrap

بحث سريع: Algolia أو Elasticsearch أو Search API مدمجة

# كيفية التشغيل
.
افتح الموقع على أي جهاز (حاسوب، جوال، أو تابلت).
استخدم شريط البحث للعثور على محتوى الدروس بسرعة.
استعرض جداول المواعيد والمحاضرات بسهولة.
اطلع على الأخبار والإعلانات الجامعية.
حمل وشارك الدروس والخرائط الذهنية.
شارك نصائحك وملخصاتك مع الطلاب الآخرين.
